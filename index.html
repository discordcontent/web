About

Welcome to QuelionBots, the platform where anyone can create bots for free! Whether you want a bot for fun, moderation, or automating tasks, we make it easy. You can customize your bots and even request advanced commands.

Features

General Features

  • Create bots easily without coding knowledge
  • Friendly and colorful interface
  • Commands for fun, moderation, and utility
  • Fully documented in the help section

Admin Commands

  • Kick, ban, mute users
  • Manage messages (delete, pin, unpin)
  • Change nicknames and roles
  • Log actions in a designated channel
  • Voice channel management (move users, mute/unmute)

Note: Admin commands only work if you have the required permissions. Contact us to add or remove commands.

Fun Commands

  • Random memes and jokes
  • Games and trivia
  • Music commands with YouTube integration
  • Profile customization including avatar upload
